## import_map.json

For a transparent usage:

```json
{
  "imports": {
    "preact": "https://esm.sh/preact@10.15.1",
    "preact/": "https://esm.sh/preact@10.15.1/",
    "react-icons/si": "https://deno.land/x/react_icons_si@1.0.0//mod.ts",
  }
}
```
